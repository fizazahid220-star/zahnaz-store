Zahnaz Next.js Starter (Tailwind)
---------------------------------
How to run:

1) Unzip the package and open terminal in the project folder.
2) Run:
   npm install
   npm run dev

3) Open http://localhost:3000

Notes:
- Tailwind is pre-configured (tailwind.config.js and postcss.config.js).
- This is a starter; replace placeholder images (in components/data) with your images in /public/images.
- Checkout uses dummy payment methods (COD, Easypaisa, JazzCash) and only simulates order submission.
